# Personal Expense Analysis Dashboard

## Overview
This project analyzes personal expense data from January 2026 to April 2026 using Microsoft Excel. The goal was to transform raw bank statement data into meaningful insights and support better financial decision-making.

## Data Source
The dataset was created from raw bank statement transactions. The data was manually cleaned, structured, and categorized into groups such as Food, Shopping, and Travel.

## Tools & Skills Used
- Microsoft Excel  
- Data Cleaning & Structuring  
- Pivot Tables & Pivot Charts  
- Data Visualization  
- Interactive Dashboard using Slicers  

## Analysis
- Category-wise expense analysis  
- Monthly spending trends  
- Weekday vs Weekend comparison  
- Need vs Want classification  

## Key Insights
- 75% of spending occurs on weekdays (routine-driven)  
- Shopping is the highest expense category (~₹17K)  
- 41% of expenses are non-essential (wants)  
- Spending peaked in March  

## Recommendations
- Set category-wise budget limits  
- Reduce non-essential spending  
- Track expenses weekly  
- Monitor high-spending months  

## Conclusion
This project demonstrates the ability to clean real-world data, perform analysis, and build interactive dashboards to extract actionable insights for better decision-making.
